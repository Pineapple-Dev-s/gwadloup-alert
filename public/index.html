Politique de confidentialité

Données collectées :

  • Compte utilisateur : email, pseudo, commune (optionnel). Mot de passe hashé par Supabase (bcrypt).
  • Signalements : titre, description, localisation GPS, photos, catégorie, priorité.
  • Analytics : pages visitées, appareil, navigateur, pays (déduit de la langue). Aucun cookie n'est utilisé. L'identifiant de session est un hash anonyme (IP + User-Agent + date), non réversible.
  • Signalements anonymes : aucune donnée personnelle n'est associée. Le champ user_id est NULL.

Utilisation des données :

  • Affichage des signalements sur la carte publique
  • Statistiques agrégées (nombre de signalements par commune, catégorie, etc.)
  • Modération automatique du contenu (détection d'insultes, reformulation via IA)
  • Classement des contributeurs (pseudo + score)

Partage des données :

  • Les signalements sont publics par nature (carte collaborative).
  • Les données ne sont jamais vendues à des tiers.
  • Les images sont hébergées sur ImgBB (service tiers).
  • La modération IA utilise l'API Groq (les textes modérés sont envoyés pour reformulation).

Vos droits (RGPD) :

  • Accès : consultez vos signalements dans votre profil.
  • Rectification : modifiez votre pseudo et commune dans les paramètres.
  • Suppression : supprimez vos signalements individuellement. Pour supprimer votre compte, contactez-nous par email.
  • Portabilité : contactez-nous pour obtenir une copie de vos données.

Durée de conservation : les données sont conservées tant que le service est actif. Les signalements résolus restent visibles pour l'historique.

Contact DPO : maxenceponche971@gmail.com

Conditions d'utilisation

En utilisant Gwadloup Alert, vous acceptez les conditions suivantes :

  • Contenu : les signalements doivent concerner des problèmes réels en Guadeloupe. Tout contenu diffamatoire, injurieux, discriminatoire ou illégal sera modéré automatiquement ou supprimé.
  • Véracité : vous vous engagez à ne pas publier de faux signalements.
  • Photos : les images uploadées ne doivent pas contenir de données personnelles identifiables (visages, plaques d'immatriculation) sauf si nécessaire au signalement.
  • Respect : les commentaires doivent rester courtois et constructifs.
  • Responsabilité : Gwadloup Alert est un outil de signalement citoyen. Nous ne sommes pas responsables des actions (ou inactions) des autorités compétentes.
  • Modération : un système automatique filtre les contenus inappropriés. Les administrateurs peuvent supprimer tout contenu jugé non conforme.
  • Gratuité : le service est et restera 100% gratuit.
