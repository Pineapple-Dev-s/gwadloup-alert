Politique de confidentialité

Données collectées :

  • Compte : email, pseudo, commune (optionnel). Mot de passe hashé (bcrypt).
  • Signalements : titre, description, localisation GPS, photos, catégorie, priorité.
  • Analytics : pages visitées, appareil, navigateur, pays. Aucun cookie. Identifiant session = hash anonyme non réversible.
  • Anonyme : aucune donnée personnelle associée (user_id = NULL).

Utilisation : affichage carte publique, statistiques agrégées, modération IA, classement contributeurs.

Partage : signalements publics, données jamais vendues, images sur ImgBB, modération via API Groq.

RGPD : accès via profil, rectification dans paramètres, suppression par signalement ou email, portabilité sur demande.

Conservation : tant que le service est actif.

Contact DPO : maxenceponche971@gmail.com

Conditions d'utilisation

En utilisant Gwadloup Alert, vous acceptez :

  • Les signalements doivent concerner des problèmes réels en Guadeloupe.
  • Contenu diffamatoire, injurieux ou illégal modéré automatiquement ou supprimé.
  • Pas de faux signalements.
  • Photos sans données personnelles identifiables sauf nécessité.
  • Commentaires courtois et constructifs.
  • Nous ne sommes pas responsables des actions des autorités.
  • Le service est et restera 100% gratuit.
