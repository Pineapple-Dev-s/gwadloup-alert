# 📋 Guide des catégories

Gwadloup Alèrt propose **31 catégories** de signalement, organisées en 9 groupes.

---

## 🛣️ Routes & Voirie

| Catégorie | Description | Exemples |
|-----------|-------------|----------|
| 🕳️ Nid de poule | Trous dans la chaussée | Route nationale, communale, départementale |
| ⚠️ Route dangereuse | Danger pour les usagers | Virage sans visibilité, route effondrée |
| 🚧 Signalisation abîmée | Panneaux endommagés | Stop arraché, direction illisible |
| 🚸 Marquage effacé | Lignes au sol disparues | Passages piétons, lignes continues |
| 🔶 Ralentisseur nécessaire | Zone nécessitant un ralentisseur | Écoles, zones résidentielles |

---

## 🚗 Véhicules & Épaves

| Catégorie | Description |
|-----------|-------------|
| 🚗 Véhicule abandonné | Voiture, moto, camion laissé à l'abandon |
| ⛵ Bateau abandonné | Épave de bateau sur terre ou en mer |

---

## 🗑️ Déchets & Pollution

| Catégorie | Description |
|-----------|-------------|
| 🗑️ Dépôt sauvage | Déchets abandonnés dans la nature |
| 🏖️ Pollution de plage | Déchets, algues toxiques, eaux usées |
| 🏞️ Pollution de rivière | Déversements, déchets dans les cours d'eau |
| 🗑️ Poubelle débordante | Conteneur plein non collecté |

---

## 💡 Éclairage & Électricité

| Catégorie | Description |
|-----------|-------------|
| 💡 Éclairage défaillant | Lampadaire éteint ou clignotant |
| ⚡ Câble exposé | Fil électrique visible, danger |

---

## 💧 Eau & Assainissement

| Catégorie | Description |
|-----------|-------------|
| 💧 Fuite d'eau | Canalisation qui fuit |
| 🌊 Inondation | Zone inondée, stagnation |
| 🚿 Problème assainissement | Égout bouché, odeurs |
| 🦟 Eau stagnante | Mare d'eau favorisant les moustiques |

---

## 🌿 Végétation

| Catégorie | Description |
|-----------|-------------|
| 🌿 Végétation envahissante | Herbes hautes, broussailles |
| 🌳 Arbre tombé | Arbre sur la route ou menaçant |
| 🌱 Espèce invasive | Plante nuisible (sargasses, etc.) |

---

## 🏗️ Infrastructures

| Catégorie | Description |
|-----------|-------------|
| 🏚️ Bâtiment endommagé | Structure dégradée |
| 🏗️ Bâtiment abandonné | Immeuble à l'abandon |
| 🚶 Trottoir abîmé | Trottoir cassé, impraticable |
| 🚧 Garde-fou manquant | Barrière de sécurité absente |

---

## 🛡️ Sécurité

| Catégorie | Description |
|-----------|-------------|
| 🛡️ Zone dangereuse | Lieu présentant un danger |
| 🚶 Passage piéton manquant | Traversée dangereuse |
| 🏫 Problème zone scolaire | Sécurité aux abords d'école |

---

## 📢 Nuisances

| Catégorie | Description |
|-----------|-------------|
| 🔊 Nuisance sonore | Bruit excessif récurrent |
| 🐕 Animaux errants | Chiens, chats en divagation |
| 🦟 Foyer à moustiques | Zone de reproduction |

---

## 📌 Autre

Pour tout ce qui ne rentre pas dans les catégories ci-dessus.

---

> **Conseil** : Choisissez toujours la catégorie la plus précise possible. Cela aide les autorités à orienter le signalement vers le bon service.
